## 💡 Por quê Programação?

**Primeiro:** programação é desafiadora pra caramba. Exige estudo, dedicação, e é uma área meritocrática de verdade. Quem entrega resultado é reconhecido. Isso me motiva.

**Segundo:** como programador consigo transformar ideias em sistemas reais que resolvem problemas de verdade. Não dependo só de ferramentas prontas, posso criar do zero.

**Terceiro:** eu gosto de criar. Sempre gostei. E programar é criar na essência.
